# Slideshow Images

This folder contains images for the homepage banner slideshow.

## Adding Images

1. Place your slideshow images in this folder
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
3. Recommended dimensions: 1600x900 pixels for best results
4. Name your files descriptively (e.g., `banner-1.jpg`, `promo-2.png`)

## Example

```
/public/slideshow/
  ├── banner-1.jpg
  ├── banner-2.png
  ├── banner-3.jpg
  └── README.md
```

The slideshow component will automatically detect and display all images in this folder in alphabetical order.

## Current Setup

- Auto-play: Enabled (5 seconds per slide)
- Navigation: Dots and arrow controls
- Responsive: Yes
- Touch/Swipe: Enabled on mobile devices
